## Installation

Users will need Node.js, Inquirer, Express, Dotenv, and MySQL2.
  
## Instructions 

Clone repository:
```
git clone <repository link>
```

Add npm dependencies:
```
npm install
```
Seed database:
```
npm run seed
```
Create tables and start server:
```
npm run start
```
